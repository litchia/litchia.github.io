# 冷门但有用的 css 特性

## Multi-column Layout  多列布局

## Writing Modes  书写模式

## aspect-ratio 宽高比

 比如设置宽高比为 16/9：

 ```css
  aspect-ratio: 16/9;
 ```

## gap  flex 和 grid 的间隔

## filter  滤镜

 比如每年的国家公祭日会把颜色调成黑白。

## backdrop-filter  毛玻璃

 ```css
  background-color: rgba(255, 255, 255, 0.3);
  backdrop-filter: blur(20px);
 ```

## accent-color  常见UI比如 input 的强调色

## will-change  会发生动画的属性
